## xbps-updater

В будущем буду добавлять все больше утилит для прекрасного пользования Void'ом.

# В планах:
- **info:** Выводить информацию о системе, без асци естесна, фастфетч кал. Ядро, кол-во пакетов, все репозитории.
- **desktop preset**: установка нужного граф окружения с удалением старого, добавление новых сервисов в автостарт.

# Как использовать?
Качаете репозиторий и либо из под директории выполняете ./update, либо пихаете /update в /usr/local/bin и легко используете тулзу через команду update.
